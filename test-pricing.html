Testing Pricing Page

Open the browser console and navigate to: http://localhost:5173/pricing

Things to check:

  1. Do you see any console errors when the page loads?
  2. Click on "Get Started" for Basic or Pro plan
  3. Does it show "Processing..." or any error?
  4. Check Network tab for failed API calls

Common Issues:

  • Button not clickable: Check if button is disabled (should only be disabled for Free plan or current plan)
  • No response on click: Check console for JavaScript errors
  • Razorpay not loading: Check if Razorpay script loads in Network tab
